Code[learn] -
Weekly Classes

Every term we'll run an eight-week course on the basics of a widely-used coding language. The classes are 1 hour long and we’ll get you coding from the first session- so bring your laptop!

Code[laborate] -
Social Coding Sessions

Come along to the social coding sessions to get help on a project, see what others are working on and code together with the comfort of some free snacks!

Resources and Code Mentors

Stuck on your coding project? Need a hand coming to grips with a particular coding topic? Just contact us and one of our amazing coding mentors will be happy to help!
